# React Todo App

A simple and clean Todo application built using **React**, **Context API**, and **LocalStorage**.  
This project demonstrates how to manage global state using React Context and persist data using browser LocalStorage.

## Features

- Add new todos
- Edit existing todos
- Mark todos as completed
- Delete todos
- Data persistence using LocalStorage
- Global state management using Context API
- Clean UI using Tailwind CSS

## Technologies Used

- React
- Context API
- JavaScript (ES6+)
- Tailwind CSS
- LocalStorage


## Run the Project Locally

Follow these steps to run the project on your local machine.

### 1. Clone the repository

git clone https://github.com/santosh312-dev/react-todo-app.git

### 2. Navigate into the project folder

cd react-todo-app

### 3. Install dependencies

npm install

### 4. Start the development server

npm run dev
